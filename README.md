# React Query Example

### Run a backend server

- change directory to `/backend`.

- If you have `npm`

```
npm install
npm run dev
```

- If you have `yarn`

```
yarn
npm dev
```

### Run a frontend server

- change directory to `/frontend`.

- If you have `npm`

```
npm install
npm run dev
```

- If you have `yarn`

```
yarn
npm dev
```
